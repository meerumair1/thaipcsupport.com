So what exactly does our product do?
Bring Your Own Cloud Account
Our unique deployment model lets you host in your own cloud account, migrate over hosting credits and leverage Reserved Instances.
Virtual Private Cloud Support
Protect your NoSQL and SQL clusters by hosting in an AWS Virtual Private Cloud (VPC) or Azure Virtual Network (VNET) subnets.
Super User Access
Keep 100% control over your clusters with full admin access to all of your MongoDB® Database, Redis™, MySQL or PostgreSQL clusters.
Full SSH Access
Keep full SSH access so that your team can install custom agents on the underlying machines.
Extension Support
Install custom plugins and extensions to accommodate all of your NoSQL and SQL needs with no restrictions.
Local SSD Support
Get high read and write SQL and NoSQL cluster throughput with special high-performance SSD-based clusters on Azure or AWS.
Hybrid Cloud Support
Deploy MongoDB® Database and Redis™ clusters On-Premise and in the Public Cloud, and manage you hybrid deployments from a single console.
24/7 Expert Support Included

SSL & Disk Encryption
